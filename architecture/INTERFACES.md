### `src/nmon/data_source/nvml.py`

## Role
Provides GPU metrics collection via NVML (NVIDIA Management Library).

## Contract: Module-level functions

### `get_gpu_metrics()`
- **Requires:** NVML must be initialized and available; GPU devices must be present and accessible
- **Guarantees:** Returns list of GPU metrics dictionaries with keys: 'gpu_id', 'utilization', 'memory_used', 'memory_total', 'temperature', 'power_draw', 'timestamp'
- **Raises:** `pynvml.NVMLError` -- if NVML operations fail or GPU devices are not accessible
- **Silent failure:** None
- **Thread safety:** Unsafe

## Module Invariants
- All GPU metrics are collected in a single atomic operation
- Timestamps are in UTC epoch seconds
- GPU IDs are integers starting from 0

## Resource Lifecycle
- NVML library is initialized on first call and kept active
- No explicit cleanup required beyond NVML's internal management
